## Getting started

```shell script
npm install

# Development
npm run dev

# Build dist
npm run build
```

## Key Features

1. Authentication via Next-Auth
2. login/register
3. articles: create, edit, delete
4. filter articles by tag, author, favorited
5. comments on articles
6. favorite articles
7. follow other users
